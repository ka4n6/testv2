Due to diversity in operating systems, file systems, encryption, hardware etc., no single tool supports all types of digital devices. We need to depend on variety of digital forensic tools (free and commercial) to achieve our goals. This is an effort to compile all available tools and frameworks making it readily available for digital forensic community. Please note that tools are not sorted based on their utility and importance. Please ignore if any tool is added to wrong category.
